ShinaYuu Music is a modified build based on the original **Mineradio** project by XxHuberrr. The original license is retained in `LICENSE`.

## Third-party libraries

- Electron
- Three.js
- GSAP
- music-tempo
- mpg123-decoder
- youtubei.js

## Third-party services

- Spotify Web API and Accounts OAuth
- YouTube public music/video services
- LRCLIB lyrics API
- Open-Meteo weather data

ShinaYuu Music is not an official client of Spotify, YouTube or LRCLIB and is not affiliated with those services. Their names, logos and trademarks belong to their respective owners.

The conversion preserves substantial original Mineradio visual design and code. Internal compatibility keys named `netease` and `qq` remain in selected UI code paths only to avoid destabilizing the original interface; they now map to Spotify and YouTube respectively.
